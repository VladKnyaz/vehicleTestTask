# Тестовое задание на React


- Редактирование цены/модели/марки
- Просмотр на карте
- Сортировка по цене и году
- Удаление

## Как запустить

- Открываем терминал
- ```cd 'путь/к/папке'```
- ```yarn``` или ```npm i```
- ```yarn dev``` или ```yarn build && yarn preview```

nodejs version - 20.11.1
